- ## Calendar Program
This is a console-based calendar program written in C.

### Logic Used
- The program takes month and year as input from the user.
- It checks whether the given year is a leap year.
- Conditional statements are used to determine the number of days in each month.
- The starting day of the month is calculated so dates align correctly.
- Input validation is applied to avoid invalid month or year values.
- The program displays a formatted calendar for the given month.

## Concepts Used
- Input/Output
- Variables
- Conditional statements
- Basic arithmetic operations

## How to Run
Compile using:
gcc filename.c

Then run:
./a.out
